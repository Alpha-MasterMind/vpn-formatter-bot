🤖 VPN Config Formatter Bot

This bot converts raw VPN configuration text (from apps like HA Tunnel, TLS Tunnel, or bots like @Decode7Bot) into clean, ready‑to‑use formats for 11+ VPN applications.

📋 What I Do:
• Parse config dumps containing [ᯤ] [Key] : Value
• Validate missing important fields (SNI, Host, Payload)
• Generate clipboard‑ready text for HA Tunnel, TLS Tunnel, HTTP Custom, HTTP Injector, NapsternetV, OpenVPN, WireGuard, Shadowsocks, V2Ray, NetMod, and HTTP Net Header
• Offer an interactive payload builder with /build
• Scan SSL certificates with /scan domain.com

🚀 How to Use:
1. Send me a config text block.
2. I'll show buttons for each supported app.
3. Tap an app to get formatted output.
4. Copy the text and paste it into your VPN app.

🔧 Commands:
/start – This welcome message
/help – List all supported apps and commands
/example – Show a sample config text
/stats – View bot usage statistics
/build – Step‑by‑step payload builder
/scan <domain> – Check SSL certificate info

✨ No files, no encryption — just fast formatting.
